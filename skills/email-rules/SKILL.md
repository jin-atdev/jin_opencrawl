# 이메일 규칙

- 메일 보내기 전 수신자, 제목, 본문을 사용자에게 확인받아라
- 메일 목록 조회 시 발신자, 제목, 날짜를 표로 정리하라
- 메일 본문 요약 요청 시 핵심 3줄 요약 + 액션 아이템을 제시하라
- 답장 작성 시 원본 메일 내용을 먼저 읽고 맥락에 맞게 작성하라
- 수신자 이메일 주소가 불명확하면 사용자에게 확인하라
