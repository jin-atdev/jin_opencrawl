# Heartbeat 체크리스트

아래 항목을 순서대로 확인하라:

1. **다가오는 일정**: list_calendar_events로 30분 내 시작하는 일정이 있는지 확인
2. **읽지 않은 긴급 메일**: list_emails로 최근 읽지 않은 메일 확인
3. **판단**: 알릴 게 있으면 간결하게 알림 메시지를 작성하라. 알릴 게 없으면 정확히 "HEARTBEAT_OK" 라고만 응답하라.

## 알림 형식
- 일정이 있으면 시간과 제목을 포함하라
- 읽지 않은 메일이 있으면 발신자와 제목을 포함하라
- 여러 항목이면 합쳐서 하나의 메시지로 전송

## 중요
- 확인만 하라. 일정 생성/수정/삭제, 메일 전송 등 변경 작업은 절대 하지 마라.
- 알릴 게 없으면 반드시 "HEARTBEAT_OK" 라고만 응답하라. 다른 인사나 설명을 붙이지 마라.
